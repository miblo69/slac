Web based upload - SLAC Webform File Acceptor

Please report any issues to info@tornado.se.
Type of Logfile to analyze

Web server (Microsoft IIS, Apache, Domino etc)
Checkpoint FW-1 (ASCII/text exported only)

All log files are preferably compressed with gzip prior to submitting them - to minimize bandwidth & time for analysis.

Log file to Upload:
Perform Reverse Name resolution  (can increase analysis time with 100-400%)
Add explanatory text to each report section (recommended for first time users).

Output Format Text - recommended. Produces legible output in your browser.
XML - for parsing the report, and possibly using your own XML Schema etc

 

Please note: There is an issue with large files (>100MB uncompressed) timing out and results not being returned in your browser. We're working on this currently. The workaround is to e-mail large log files to logger@a51.mine.nu (web) or fwlogger@a51.mine.nu (FW-1).
